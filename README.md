# roma9-data-plattorm
Personal data platform for lifelong analytics: pipelines, dbt models, NSA, NLP, LLM, dashboards and experimentation.
Este repositório é a espinha dorsal da minha plataforma pessoal de dados — desde pipelines, modelagem com dbt, análises sociais (NSA), processamento de linguagem natural (NLP), até dashboards e experimentos.

## Estrutura principal

- **data-lake/** – arquivos brutos e refinados
- **dbt/** – modelos dbt para transformação e DWH
- **analytics/** – análises por domínio
- **notebooks/** – notebooks exploratórios
- **scripts/** – scripts de ingestão e utilitários

## Objetivos

1. Construir um **data lake** escalável e barato
2. Criar **pipelines e transformações** com dbt
3. Fazer análises de redes sociais (NSA) de votações e padrões
4. Aplicar **NLP** em discursos políticos e outros textos
5. Visualizar resultados em dashboards
6. Manter um laboratório de dados para projetos futuros

## Tecnologias

- Git + GitHub para versionamento
- GCP (Cloud Storage + BigQuery)
- dbt (transformações e documentação)
- Python (ETL, NLP, NSA)
- Looker Studio / Power BI (dashboards)
